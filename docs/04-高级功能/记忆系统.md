# 🧠 记忆系统

## 概述

OpenClaw 的记忆系统让你能够跨会话记住重要信息，包括用户偏好、对话历史和自定义数据。

## 记忆类型

### 1. 短期记忆 (Session)

当前会话内的记忆，会话结束后清除：

```javascript
// 设置短期记忆
session.set("currentTask", "写代码");

// 读取
const task = session.get("currentTask");
```

### 2. 长期记忆 (Memory)

持久化存储，跨会话保留：

```javascript
// 设置长期记忆
memory.set("user:123:preference", { theme: "dark" });

// 读取
const pref = memory.get("user:123:preference");
```

### 3. 每日记忆 (Daily)

按日期组织的记忆，便于回顾：

```javascript
// 记录今天的事情
memory.today().add("important", "杰哥喜欢简洁的回答");

// 查看今天记录
memory.today().get("important");
```

## 使用方法

### 保存用户偏好

```javascript
// 记住用户名称
await memory.setUserPref("name", "杰哥");

// 记住用户偏好
await memory.setUserPref("language", "中文");
await memory.setUserPref("tone", "简洁");
await memory.setUserPref("notifications", true);

// 获取偏好
const name = await memory.getUserPref("name");
```

### 记住重要信息

```javascript
// 记住重要约定
await memory.remember("meeting", {
  type: "meeting",
  with: "张三",
  time: "2026-03-05 14:00",
  topic: "项目讨论"
});

// 搜索记忆
const meetings = await memory.search("meeting");
```

### 记住学习内容

```javascript
// 记录学习心得
await memory.learn("lesson", {
  topic: "OpenClaw",
  insight: "多渠道集成很方便",
  date: new Date().toISOString()
});
```

## 记忆策略

### 自动记忆

```javascript
export default {
  name: "auto-memory",
  
  async onMessage(message) {
    // 自动记住用户提到的信息
    if (message.content.includes("我叫")) {
      const name = message.content.match(/我叫(.+)/)[1];
      await memory.setUserPref("name", name);
    }
    
    // 记住用户兴趣
    if (message.content.includes("喜欢")) {
      await memory.addUserInterest(message.from, extractInterest(message.content));
    }
  }
}
```

### 记忆摘要

```javascript
// 定期生成记忆摘要
export default {
  schedule: "0 0 * * *", // 每天午夜
  
  async onScheduled() {
    const summary = await memory.summarize();
    await memory.set("daily_summary", summary);
  }
}
```

### 遗忘机制

```javascript
// 清理过期记忆
export default {
  schedule: "0 2 * * 0", // 每周日凌晨
  
  async onScheduled() {
    // 删除 30 天前的短期记忆
    await memory.forgetOlderThan(30 * 24 * 60 * 60 * 1000);
    
    // 删除临时数据
    await memory.clearTemp();
  }
}
```

## 最佳实践

### 1. 定期整理

```javascript
// 每周回顾
export default {
  schedule: "0 10 * * 0", // 每周日上午
  
  async onScheduled() {
    await memory.consolidate(); // 整理记忆
    await memory.cleanup();     // 清理无用记忆
  }
}
```

### 2. 重要标记

```javascript
// 标记重要记忆（不会被清理）
await memory.set("important:meeting", data, { important: true });

// 标记需要复习
await memory.set("review:concept", data, { review: true });
```

### 3. 隐私保护

```javascript
// 敏感信息加密
await memory.setSecure("secret:token", encryptedValue);

// 只在必要时访问
const sensitive = await memory.getSecure("secret:token");
```

## 导出与备份

### 导出记忆

```bash
# 导出所有记忆
openclaw memory export --format json

# 导出特定用户
openclaw memory export --user user123

# 导出时间范围
openclaw memory export --from 2026-01-01 --to 2026-03-01
```

### 导入记忆

```bash
# 从文件导入
openclaw memory import backup.json

# 合并导入（不覆盖）
openclaw memory import backup.json --merge
```

## 调试记忆

### 查看记忆状态

```bash
# 查看记忆统计
openclaw memory stats

# 查看特定键
openclaw memory get "key"

# 查看用户记忆
openclaw memory user user123
```

### 清除记忆

```bash
# 清除所有记忆
openclaw memory clear --force

# 清除特定用户
openclaw memory clear --user user123

# 清除特定日期
openclaw memory clear --date 2026-03-01
```
