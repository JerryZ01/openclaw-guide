# 🛠️ 技能开发

## 什么是技能？

技能（Skill）是 OpenClaw 的扩展机制，让你可以自定义消息处理逻辑、添加新功能。

## 技能结构

```
skills/
└── my-skill/
    ├── SKILL.md        # 技能描述
    ├── index.js        # 入口文件
    └── config.json     # 配置（可选）
```

## 创建技能

### 1. 创建目录

```bash
mkdir -p ~/.openclaw/workspace/skills/my-skill
```

### 2. 编写技能代码

```javascript
// ~/.openclaw/workspace/skills/my-skill/index.js

export default {
  // 技能名称
  name: "my-skill",
  
  // 技能版本
  version: "1.0.0",
  
  // 技能描述
  description: "我的第一个技能",
  
  // 消息处理规则
  rules: [
    {
      // 匹配规则
      pattern: /hello|你好/,
      // 回复内容
      response: "你好！我是小二 🦊"
    },
    {
      // 使用 AI 回复
      pattern: /.*/,
      ai: true,
      system: "你是一个有帮助的助手，用中文回复"
    }
  ],
  
  // 生命周期钩子
  async onLoad() {
    console.log("技能加载完成");
  },
  
  async onUnload() {
    console.log("技能卸载");
  }
}
```

## 规则类型

### 关键词匹配

```javascript
{
  pattern: /帮助/,
  response: "你可以问我：\n1. 天气\n2. 新闻\n3. 其他"
}
```

### 正则表达式

```javascript
{
  // 匹配特定格式
  pattern: /^(\d{4})-(\d{2})-(\d{2})$/,
  response: (match) => {
    const [_, year, month, day] = match;
    return `日期：${year}年${month}月${day}日`;
  }
}
```

### AI 对话

```javascript
{
  // 所有消息都用 AI 处理
  pattern: /.*/,
  ai: true,
  system: "你是一个友好的助手"
}
```

### 回调函数

```javascript
{
  // 自定义处理逻辑
  handler: async (message) => {
    if (message.content.includes("天气")) {
      return await getWeather();
    }
    return null;
  }
}
```

```json

{
  "enabled": true,
  "priority": 10,
  "settings": {
    "apiKey": "xxx",
    "option": "value"
```

### 全局启用技能

```bash
# 列出技能
openclaw skills list

# 启用技能
openclaw skills enable my-skill

# 禁用技能
openclaw skills disable my-skill
```

## 高级功能

### 消息拦截

```javascript
export default {
  name: "interceptor",
  
  // 在消息处理前执行
  async onBeforeMessage(message) {
    // 可以修改消息
    message.preprocessed = true;
    return message;
  },
  
  // 在消息处理后执行
  async onAfterMessage(message, response) {
    // 可以修改回复
    response.signature = " - 小二";
    return response;
  }
}
```

### 定时任务

```javascript
export default {
  name: "scheduled-task",
  
  // Cron 表达式
  schedule: "0 9 * * *",
  
  async onScheduled() {
    // 每天早上 9 点执行
    await this.sendMessage({
      channel: "telegram",
      target: "my-chat",
      message: "早上好！"
    });
  }
}
```

### HTTP 端点

```javascript
export default {
  name: "webhook-skill",
  
  // 注册 HTTP 端点
  endpoints: [
    {
      path: "/webhook",
      method: "POST",
      handler: async (req, res) => {
        const { message } = req.body;
        return await this.processMessage(message);
      }
    }
  ]
}
```

### 工具注册

```javascript
export default {
  name: "tool-skill",
  
  // 注册自定义工具
  tools: [
    {
      name: "get_weather",
      description: "获取天气信息",
      parameters: {
        type: "object",
        properties: {
          city: { type: "string" }
        },
        required: ["city"]
      },
      handler: async ({ city }) => {
        return await fetchWeather(city);
      }
    }
  ]
}
```

## 技能市场

### 发布技能

1. 创建 GitHub 仓库
2. 添加 `SKILL.md` 描述
3. 提交到社区

### 安装技能

```bash
# 从 GitHub 安装
openclaw skills install github.com/user/repo

# 从本地安装
openclaw skills install ./my-skill
```

## 最佳实践

1. **单一职责**：每个技能只做一件事
2. **错误处理**：做好异常捕获
3. **配置分离**：敏感信息放环境变量
4. **日志记录**：便于调试和问题排查
5. **版本管理**：做好版本控制
