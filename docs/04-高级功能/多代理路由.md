# 🔀 多代理路由

## 概念

多代理路由允许你根据不同条件（用户、渠道、消息内容）将请求路由到不同的 AI 代理或工作空间。

## 基本配置

### 默认代理配置

```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "anthropic/claude-3.5-sonnet"
      },
      "maxTokens": 8192
    }
  }
}
```

## 条件路由

### 按用户路由

```javascript
// skills/router/index.js
export default {
  name: "router",
  async onMessage(message) {
    const { from, channel } = message;
    
    // 根据用户 ID 路由到不同代理
    if (from === "user1") {
      return await this.delegateTo("claude-agent", message);
    } else if (from === "user2") {
      return await this.delegateTo("gpt-agent", message);
    }
    
    // 默认代理
    return await this.delegateTo("default", message);
  }
}
```

### 按渠道路由

```javascript
// 根据渠道选择模型
export default {
  name: "channel-router",
  async onMessage(message) {
    const { channel } = message;
    
    switch (channel) {
      case "whatsapp":
        // WhatsApp 用轻量模型
        return await this.delegateTo("fast-model", message);
      case "telegram":
        // Telegram 用标准模型
        return await this.delegateTo("standard-model", message);
      case "feishu":
        // 飞书用中文优化模型
        return await this.delegateTo("chinese-model", message);
      default:
        return await this.delegateTo("default", message);
    }
  }
}
```

### 按消息内容路由

```javascript
// 根据关键词路由
export default {
  name: "content-router",
  async onMessage(message) {
    const { content } = message;
    
    // 技术问题
    if (content.includes("代码") || content.includes("bug")) {
      return await this.delegateTo("developer-agent", message);
    }
    
    // 业务问题
    if (content.includes("订单") || content.includes("支付")) {
      return await this.delegateTo("support-agent", message);
    }
    
    // 默认
    return await this.delegateTo("general-agent", message);
  }
}
```

## 隔离会话

### main 会话

主会话共享状态，适合长期对话：

```javascript
// 在主会话中
await sessions.send("main", {
  message: "继续之前的任务"
});
```

### isolated 会话

隔离会话独立运行，适合独立任务：

```javascript
// 创建隔离会话
const session = await sessions.create({
  type: "isolated",
  model: "claude-3.5-sonnet",
  context: "你是一个代码审查助手"
});
```

## 高级路由规则

### 负载均衡

```javascript
// 轮询选择代理
let currentIndex = 0;
const agents = ["agent-1", "agent-2", "agent-3"];

export default {
  name: "load-balancer",
  async onMessage(message) {
    const agent = agents[currentIndex];
    currentIndex = (currentIndex + 1) % agents.length;
    
    return await this.delegateTo(agent, message);
  }
}
```

### 故障转移

```javascript
// 主代理失败时切换到备用
export default {
  name: "failover",
  async onMessage(message) {
    try {
      return await this.delegateTo("primary-agent", message);
    } catch (error) {
      console.log("Primary failed, trying backup...");
      return await this.delegateTo("backup-agent", message);
    }
  }
}
```

### 限流路由

```javascript
// 基于用户限流
const userRates = new Map();

export default {
  name: "rate-limiter",
  async onMessage(message) {
    const user = message.from;
    const now = Date.now();
    const window = 60 * 1000; // 1 分钟
    
    // 检查速率
    if (!userRates.has(user)) {
      userRates.set(user, []);
    }
    
    const requests = userRates.get(user).filter(t => now - t < window);
    
    if (requests.length >= 10) {
      return "请求过于频繁，请稍后再试";
    }
    
    requests.push(now);
    userRates.set(user, requests);
    
    return await this.delegateTo("default", message);
  }
}
```

## 消息路由示例

### 路由到不同渠道

```javascript
// 跨渠道转发
export default {
  name: "cross-channel",
  async onMessage(message) {
    // 在 Telegram 收到，转发到飞书
    if (message.channel === "telegram") {
      await this.sendToChannel("feishu", {
        target: "ou_xxx",
        message: `来自 Telegram: ${message.content}`
      });
    }
    
    return "已转发";
  }
}
```

### 消息队列

```javascript
// 消息队列处理
const queue = [];

export default {
  name: "queue-processor",
  async onMessage(message) {
    // 加入队列
    queue.push(message);
    
    // 异步处理
    this.processQueue();
    
    return "消息已收到，正在处理...";
  },
  
  async processQueue() {
    while (queue.length > 0) {
      const msg = queue.shift();
      await this.delegateTo("worker-agent", msg);
    }
  }
}
```

```javascript
// 路由统计
const stats = {
  total: 0,
  byAgent: {},
  byChannel: {}
};

export default {
  name: "stats-collector",
  async onMessage(message) {
    stats.total++;
    stats.byAgent[message.agent] = (stats.byAgent[message.agent] || 0) + 1;
    stats.byChannel[message.channel] = (stats.byChannel[message.channel] || 0) + 1;
    
    return await this.delegateTo(message.agent, message);
  }
}
```

## 最佳实践

1. **清晰分离职责**：每个代理负责特定场景
2. **设置超时**：防止代理卡住
3. **错误处理**：优雅降级到备用代理
4. **监控路由**：追踪代理使用情况
5. **定期优化**：根据使用情况调整路由规则
