## 2. 自动写周报

### 功能
- 读取一周内所有聊天/邮件/代码提交
- 自动生成周报
- 可定制格式

### 实现

```javascript
// skills/weekly-report/index.js
module.exports = {
  name: "weekly-report",
  version: "1.0.0",
  
  schedule: "0 18 * * 5", // 周五下午6点
  
  async onScheduled() {
    // 1. 获取一周的数据
    const emails = await this.getEmails('thisweek');
    const commits = await this.getGitCommits('thisweek');
    const messages = await this.getChatMessages('thisweek');
    const tasks = await this.getTaskChanges('thisweek');
    
    // 2. AI 分析生成周报
    const report = await this.generateReport({
      emails,
      commits,
      messages,
      tasks
    });
    
    // 3. 发送周报
    await this.sendReport(report);
  },
  
  async generateReport(data) {
    return await this.ai.chat(`
      生成一份周报，包含：
      
      1. 本周完成的工作 (从 ${data.commits.length} 个提交和 ${data.tasks.length} 个任务)
      2. 收到的邮件数量: ${data.emails.length}
      3. 聊天消息数量: ${data.messages.length}
      4. 下周计划
      
      用简洁的格式返回。
    `);
  }
};
```

### 输出示例

```
📊 周报 - 2026年2月24日-2月28日

✅ 完成
- 完成用户认证模块开发
- 修复登录页面 bug
- 代码审查 12 个 PR

📧 邮件
- 收到 45 封邮件
- 处理 8 封重要邮件

💬 沟通
- 群聊消息 120+ 条
- 私聊 35 条

🎯 下周计划
- 完成支付模块
- 性能优化
- 开始测试
```

## 3. AI 相亲匹配

### 功能
- 读取用户资料
- AI 分析匹配度
- 帮你"右滑"

⚠️ 注意：仅供娱乐，使用有风险！

### 实现思路

```javascript
// skills/dating-assistant/index.js
module.exports = {
  name: "dating-assistant",
  version: "1.0.0",
  
  // 读取 Tinder/Bumble 个人资料
  async analyzeProfile(profile) {
    // AI 评估匹配度
    const score = await this.ai.match(profile, this.userPrefs);
    
    if (score > 80) {
      return { action: 'like', score };
    } else if (score > 60) {
      return { action: 'superlike', score };
    }
    return { action: 'pass', score };
  }
};
```

## 4. AI 写作助手

### 功能
- 帮你写文章/邮件
- 优化文案
- 检查语法

### 配置

```json
{
  "skills": {
    "writing-assistant": {
      "enabled": true,
      "modes": ["improve", "shorten", "expand", "formal", "casual"],
      "defaultMode": "improve"
    }
  }
}
```

### 使用

```
用户: 帮我把这封信改得更正式
OpenClaw: [AI 改写后的正式版本]
```

      "enabled": true,
      "wakeWord": "Hey OpenClaw",
      "channels": ["telegram", "whatsapp"],
      "voice": {
        "enabled": true,
        "language": "zh-CN"
      }
    }
  }
}
