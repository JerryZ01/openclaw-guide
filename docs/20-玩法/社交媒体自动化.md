## 1. 自动发 LinkedIn/Twitter

### 功能
- 抓取行业资讯/新闻
- AI 生成适合平台的文案
- 自动定时发布

### 配置示例

```json
{
  "skills": {
    "social-poster": {
      "enabled": true,
      "platforms": ["linkedin", "twitter"],
      "schedule": "0 9 * * 1-5",
      "sources": [
        "https://news.ycombinator.com",
        "https://techcrunch.com"
      ],
      "prompt": "你是一个科技博主，擅长写有趣的科技动态推文"
    }
  }
}
```

### 实现代码

```javascript
// skills/social-poster/index.js
module.exports = {
  name: "social-poster",
  version: "1.0.0",
  
  schedule: "0 9 * * 1-5", // 工作日早上9点
  
  async onScheduled() {
    // 1. 获取资讯
    const news = await this.gatherNews();
    
    // 2. 生成文案
    const posts = await Promise.all([
      this.generatePost(news, 'twitter'),
      this.generatePost(news, 'linkedin')
    ]);
    
    // 3. 发布
    for (const post of posts) {
      await this.postToPlatform(post);
    }
  },
  
  async gatherNews() {
    const sources = [
      'https://news.ycombinator.com',
      'https://techcrunch.com'
    ];
    
    const news = [];
    for (const url of sources) {
      const articles = await this.fetchArticles(url);
      news.push(...articles.slice(0, 5));
    }
    return news;
  },
  
  async generatePost(news, platform) {
    const prompt = platform === 'twitter' 
      ? "生成一条140字以内的科技推文，语气有趣"
      : "生成一段专业的LinkedIn动态，200字以内";
    
    return await this.ai.chat(
      `从以下新闻中生成一条${platform}帖子：\n${JSON.stringify(news)}`,
      { system: prompt }
    );
  },
  
  async postToPlatform(post) {
    // 根据平台调用不同API
    await this.twitter.post(post.content);
    await this.linkedin.post(post.content);
  }
};
```

## 2. YouTube 数据分析

### 功能
- 追踪上百个视频的播放数据
- 自动生成周报/月报
- 异常数据告警

### 配置

```json
{
  "skills": {
    "youtube-analytics": {
      "enabled": true,
      "channelId": "YOUR_CHANNEL_ID",
      "schedule": "0 10 * * 0", // 每周日
      "metrics": ["views", "likes", "comments", "watchTime"],
      "notify": {
        "channel": "feishu",
        "target": "YOUR_USER_ID"
      }
    }
  }
}
```

### 实现

```javascript
// skills/youtube-analytics/index.js
module.exports = {
  name: "youtube-analytics",
  
  async onScheduled() {
    // 获取本周数据
    const stats = await this.youtube.getChannelStats();
    
    // 生成报告
    const report = this.generateReport(stats);
    
    // 发送报告
    await this.sendReport(report);
  },
  
  generateReport(stats) {
    const totalViews = stats.videos.reduce((sum, v) => sum + v.views, 0);
    const topVideo = stats.videos.sort((a, b) => b.views - a.views)[0];
    
    return `📊 YouTube 周报

总播放: ${totalViews.toLocaleString()}
新增订阅: ${stats.newSubscribers}
最热视频: ${topVideo.title}
播放: ${topVideo.views.toLocaleString()}

📈 趋势: ${stats.viewsChange > 0 ? '↑' : '↓'} ${Math.abs(stats.viewsChange)}%`;
  }
};
```

## 3. 多平台一键发布

## 4. Reddit 营销自动化

### 功能
- 自动搜索相关subreddit
- 监控关键词
- 自动回复/评论
- 引流到目标地址

### 注意事项
- ⚠️ 遵守 Reddit 规则，避免被封号
- 建议设置合理的请求频率
- 优先提供价值，再适当引流

## 常见问题

### Q: 发帖频率怎么控制？
A: 建议每天每个平台不超过 3-5 条，避免被封禁。

### Q: 内容原创度怎么保证？
A: 用 AI 改写+添加个人见解，不要纯搬运。

### Q: 需要配置哪些 API？
A:
- Twitter/X: Developer Portal
- LinkedIn: Marketing API
- YouTube: Data API v3
- Instagram: Facebook Graph API
