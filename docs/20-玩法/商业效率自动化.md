## 1. 邮件自动处理

> OpenClaw 最火的功能！被称为"杀手级应用"

### 功能
- 自动分类邮件（重要/普通/垃圾）
- 自动回复常见问题
- 提取关键信息生成摘要
- 转发到指定渠道

### 配置

```json
{
  "skills": {
    "email-assistant": {
      "enabled": true,
      "gmail": {
        "enabled": true,
        "watchLabels": ["INBOX", "IMPORTANT"]
      },
      "rules": [
        {
          "match": "from:hr@company.com",
          "action": "label:HR",
          "notify": true
        },
        {
          "match": "subject:订单",
          "action": "label:订单",
          "forward": "sales@company.com"
        }
      ],
      "autoReply": {
        "enabled": true,
        "keywords": {
          "请假": "您的请假申请已转交HR处理...",
          "报销": "报销流程请参考公司Wiki..."
        }
      }
    }
  }
}
```

### 实现

```javascript
// skills/email-assistant/index.js
module.exports = {
  name: "email-assistant",
  version: "1.0.0",
  
  // 每小时检查一次
  schedule: "0 * * * *",
  
  async onScheduled() {
    const emails = await this.gmail.getUnread();
    
    for (const email of emails) {
      // 分类
      const category = this.categorize(email);
      
      // 提取关键信息
      const summary = await this.summarize(email);
      
      // 根据类别处理
      if (category === 'important') {
        await this.notifyUser(email, summary);
      } else if (category === 'auto-reply') {
        await this.autoReply(email);
      }
      
      // 标记已读/归档
      await this.gmail.markAsRead(email.id);
      await this.gmail.addLabel(email.id, category);
    }
  },
  
  categorize(email) {
    const keywords = ['urgent', '紧急', 'ASAP'];
    if (keywords.some(k => email.subject.includes(k))) {
      return 'important';
    }
    return 'normal';
  },
  
  async summarize(email) {
    return await this.ai.chat(
      `用一句话总结这封邮件的核心内容：${email.subject}\n${email.body}`
    );
  }
};
```

## 2. CEO 级别仪表盘

### 功能
- 多代理分工协作
- 一个看邮件、一个看日程、一个做汇总
- 决策建议生成

### 架构

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  邮件代理    │ -> |  日程代理   │ -> |  汇总代理   │
│  (Email)    │    │  (Calendar) │    │  (Summary)  │
└─────────────┘    └─────────────┘    └─────────────┘
       │                  │                  │
       └──────────────────┼──────────────────┘
                          │
                    ┌─────▼─────┐
                    │  CEO 简报  │
                    └───────────┘
```

### 配置

```json
{
  "model": {
    "agents": {
      "email-agent": {
        "role": "邮件分析师",
        "tasks": ["分类", "摘要", "优先级排序"]
      },
      "calendar-agent": {
        "role": "日程管理器",
        "tasks": ["冲突检测", "时间优化", "会议准备"]
      },
      "summary-agent": {
        "role": "决策助手",
        "tasks": ["汇总分析", "行动建议", "风险提示"]
      }
    },
    "router": {
      "enabled": true,
      "coordination": "sequential"
    }
  }
}
```

## 3. 订阅续费监控

### 功能
- Stripe 支付失败时自动触发 3 步追回
- 监控即将到期的订阅
- 续费提醒

### 自动化流程

```
支付失败 -> 1. 发送提醒邮件
          -> 2. 3天后仍未付款 -> 发送优惠码
          -> 3. 7天后仍未付款 -> 转人工处理
```

### 配置

```json
{
  "skills": {
    "subscription-recovery": {
      "enabled": true,
      "webhook": "stripe",
      "steps": [
        {
          "name": "reminder",
          "delay": "0",
          "action": "send_email",
          "template": "payment_failed_reminder"
        },
        {
          "name": "offer",
          "delay": "3d",
          "action": "send_email",
          "template": "payment_failed_offer",
          "vars": { "discount": "20%" }
        },
        {
          "name": "escalate",
          "delay": "7d",
          "action": "notify_sales",
          "forward": "sales@company.com"
        }
      ]
    }
  }
}
```

## 4. 自动订票/酒店

### 功能
- 从邮件/日历读取出差行程
- 自动搜索并比较价格
- 发送确认链接待用户确认后下单

### 实现

```javascript
// skills/booking-assistant/index.js
module.exports = {
  name: "booking-assistant",
  
  async onScheduled() {
    // 1. 读取即将到来的行程
    const trips = await this.getUpcomingTrips();
    
    for (const trip of trips) {
      if (trip.needsBooking) {
        // 2. 搜索选项
        const flights = await this.searchFlights(trip);
        const hotels = await this.searchHotels(trip);
        
        // 3. 生成推荐
        const recommendation = await this.recommend(flights, hotels);
        
        // 4. 发送给用户确认
        await this.sendConfirmation(trip.id, recommendation);
      }
    }
  },
  
  async recommend(flights, hotels) {
    return await this.ai.chat(`
      根据以下选项生成最佳推荐：
      
      航班：${JSON.stringify(flights)}
      酒店：${JSON.stringify(hotels)}
      
      考虑：价格、位置、评分
    `);
  }
};
```
