# 🚀 快速参考卡

## 常用命令速查

### 安装与启动
```bash
npm install -g openclaw@latest    # 安装
openclaw --version               # 版本
openclaw onboard                 # 初始化
openclaw gateway --port 16784   # 启动
```

### Gateway 管理
```bash
openclaw gateway start           # 启动
openclaw gateway stop            # 停止
openclaw gateway restart         # 重启
openclaw gateway status          # 状态
openclaw logs                    # 日志
openclaw doctor                  # 诊断
```

### 渠道
```bash
openclaw channels list           # 列表
openclaw channels login wa       # 登录 WhatsApp
openclaw config set channels.telegram.enabled true
```

### 消息
```bash
openclaw message send --channel telegram --target @xxx --message "Hi"
```

### 配置
```bash
openclaw config get               # 查看
openclaw config set key value    # 设置
openclaw config unset key        # 删除
```

### 定时任务
```bash
openclaw cron list               # 列表
openclaw cron add --name "xxx" --schedule "0 9 * * *" --message "xxx"
openclaw cron remove <jobId>
```

### 会话
```bash
openclaw sessions list           # 列表
openclaw sessions history <id>  # 历史
```

## 快捷设置

### Telegram
```bash
openclaw config set channels.telegram.botToken "xxx"
openclaw config set channels.telegram.enabled true
```

### 飞书
```bash
openclaw config set channels.feishu.appId "cli_xxx"
openclaw config set channels.feishu.appSecret "xxx"
openclaw config set channels.feishu.verificationToken "xxx"
openclaw config set channels.feishu.enabled true
```

### 企业微信
```bash
openclaw config set channels.wecom.corpId "wwxxx"
openclaw config set channels.wecom.corpSecret "xxx"
openclaw config set channels.wecom.agentId 1000001
openclaw config set channels.wecom.enabled true
```
