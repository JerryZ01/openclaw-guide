# 🚀 OpenClaw 快速开始指南

本指南将帮助你在 5 分钟内完成 OpenClaw 的基础配置。

## 环境要求

- Node.js 22+
- npm 或 yarn
- 支持的操作系统：Linux、macOS、Windows (WSL)

## 安装步骤

### 1. 安装 OpenClaw CLI

```bash
# 全局安装
npm install -g openclaw@latest

# 验证安装
openclaw --version
```

### 2. 初始化配置

```bash
# 启动交互式引导
openclaw onboard
```

引导流程会询问：
- 模型选择（推荐 Anthropic Claude）
- 渠道选择（WhatsApp、Telegram 等）
- 工作目录设置

### 3. 启动 Gateway

#### 方式一：前台运行

```bash
openclaw gateway --port 16784
```

#### 方式二：后台服务运行

```bash
# 安装为系统服务
openclaw onboard --install-daemon

# 启动服务
openclaw gateway start
```

### 4. 验证服务

```bash
# 检查状态
openclaw status

# 查看日志
openclaw logs --follow
```

## 接入渠道

### WhatsApp

```bash
# 扫码登录
openclaw channels login whatsapp
```

会显示二维码，用手机 WhatsApp 扫描即可。

### Telegram

```bash
# 设置 Bot Token
openclaw config set channels.telegram.botToken "YOUR_BOT_TOKEN"

# 启动 Telegram 渠道
openclaw channels enable telegram
```

获取 Bot Token：
1. @BotFather
2. /newbot
3. 获取 Token

### 飞书

```bash
# 配置飞书应用
openclaw config set channels.feishu.appId "APP_ID"
openclaw config set channels.feishu.appSecret "APP_SECRET"
openclaw config set channels.feishu.verificationToken "VERIFICATION_TOKEN"
```

## 发送第一条消息

```bash
# 通过 CLI 发送
openclaw message send --channel telegram --target @your_username --message "你好！"
```

## 访问控制面板

启动 Gateway 后，访问：

```
http://127.0.0.1:16784/
```

或通过局域网IP访问：

```
http://<你的IP>:16784/
```

## 基础配置

配置文件位置：`~/.openclaw/openclaw.json`

```json
{
  "gateway": {
    "port": 16784,
    "bind": "0.0.0.0"
  },
  "channels": {
    "whatsapp": {
      "enabled": true
    },
    "telegram": {
      "enabled": true,
      "botToken": "your-token"
    }
  }
}
```

## 常见问题

### Q: 端口被占用？

```bash
# 查找占用进程
lsof -i :16784

# 杀掉进程
kill -9 <PID>

# 或使用 --force 强制启动
openclaw gateway --force
```

### Q: 渠道登录失败？

```bash
# 查看详细日志
openclaw logs --level debug

# 运行诊断
openclaw doctor
```

### Q: 如何后台运行？

```bash
# 使用 nohup
nohup openclaw gateway --port 16784 > /tmp/openclaw.log 2>&1 &

# 或使用 systemd（推荐）
openclaw onboard --install-daemon
```
