# 🔐 权限与安全配置

## 概述

本文档介绍 OpenClaw 的安全机制和最佳配置。

## 认证机制

### Token 认证

```json
{
  "gateway": {
    "auth": {
      "token": "your-secure-token"
    }
  }
}
```

所有 API 调用需要携带 Token：

```bash
curl -H "Authorization: Bearer your-token" \
     http://localhost:16784/api/status
```

### 生成安全 Token

```bash
# 使用 openssl
openssl rand -hex 32

# 使用 uuid
uuidgen

# 使用 Node.js
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```

## 网络安全

### 绑定地址

```json
{
  "gateway": {
    "bind": "127.0.0.1",    // 仅本地
    "bind": "192.168.1.100", // 仅内网
    "bind": "0.0.0.0"        // 全部
  }
}
```

### 防火墙

```bash
# UFW
sudo ufw allow 16784/tcp

# iptables
sudo iptables -A INPUT -p tcp --dport 16784 -s 192.168.0.0/16 -j ACCEPT
sudo iptables -A INPUT -p tcp --dport 16784 -j DROP
```

### HTTPS 配置

使用 Nginx 反向代理：

```nginx
server {
    listen 443 ssl http2;
    server_name openclaw.example.com;
    
    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;
    
    location / {
        proxy_pass http://127.0.0.1:16784;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
    }
}
```

## 数据安全

### 敏感信息加密

```javascript
// 使用 crypto 模块
const crypto = require('crypto');

function encrypt(text, key) {
  const iv = crypto.randomBytes(16);
  const cipher = crypto.createCipheriv('aes-256-cbc', Buffer.from(key, 'hex'), iv);
  let encrypted = cipher.update(text, 'utf8', 'hex');
  encrypted += cipher.final('hex');
  return iv.toString('hex') + ':' + encrypted;
}
```

### 备份加密

```bash
# 加密备份
tar czf - data/ | openssl enc -aes-256-cbc -salt -out backup.tar.gz.enc

# 解密恢复
openssl enc -d -aes-256-cbc -in backup.tar.gz.enc | tar xzf -
```

## 审计日志

```bash

openclaw logs | grep "login"

# 查看配置变更
openclaw logs | grep "config"
```

## 最佳实践

1. **使用强 Token** - 至少 32 位随机字符
2. **限制端口** - 仅内网访问
3. **启用白名单** - 明确允许的用户
4. **定期更新** - 保持 OpenClaw 最新
5. **监控日志** - 及时发现异常
6. **备份配置** - 防止配置丢失
7. **分离环境** - 开发/生产分开
