## Anthropic Claude 配置

### 获取 API Key

1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 创建 API Key
3. 复制保存（只显示一次）

### 环境变量配置

```bash
# .env
ANTHROPIC_API_KEY=sk-ant-api03-xxx
```

### 配置文件

```json
{
  "model": {
    "provider": "anthropic",
    "model": "claude-4-sonnet-20250514",
    "maxTokens": 4096,
    "temperature": 0.7
  }
}
```

### 可用模型选择

| 模型 | 推荐场景 |
|------|---------|
| `claude-4-opus-20250514` | 复杂推理、代码生成 |
| `claude-4-sonnet-20250514` | 日常对话、平衡选择 |
| `claude-4-haiku-20250514` | 简单任务、快速响应 |
| `claude-3-5-sonnet-20240229` | 稳定兼容 |

## OpenAI 配置

### 获取 API Key

1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 进入 API Keys 页面
3. 创建新 Key

### 环境变量

```bash
# .env
OPENAI_API_KEY=sk-proj-xxx
```

### 配置文件

```json
{
  "model": {
    "provider": "openai",

    "maxTokens": 4096,
    "temperature": 0.7
```

## 本地模型 (Ollama)

### 安装 Ollama

```bash
# macOS
brew install ollama

# Linux
curl -fsSL https://ollama.ai/install.sh | sh

# Docker
docker run -d -v ollama:/root/.ollama -p 11434:11434 --name ollama ollama/ollama
```

### 下载模型

```bash
# 查看可用模型
ollama list

# 下载模型
ollama pull qwen2:1.5b      # 轻量
ollama pull qwen2:3b        # 中等
ollama pull llama3.2:3b     # Llama
ollama pull phi3:14b        # Microsoft
```

### 配置文件

```json
{
  "model": {
    "provider": "ollama",
    "model": "qwen2:1.5b",
    "url": "http://localhost:11434"
  }
}
```

### 推荐配置

| 模型 | 内存要求 | 速度 | 适用场景 |
|------|---------|------|---------|
| qwen2:0.5b | 2GB | 很快 | 简单对话 |
| qwen2:1.5b | 4GB | 快 | 日常任务 |
| qwen2:3b | 6GB | 中 | 复杂任务 |
| llama3.2:3b | 6GB | 中 | 英文为主 |

## 多模型配置

### 模型优先级/降级

```json
{
  "model": {
    "primary": "claude-4-sonnet-20250514",
    "fallback": [
      {
        "model": "claude-4-haiku-20250514",
        "trigger": "primary_error"
      },
      {
        "model": "gpt-4o-mini-2024-07-18",
        "trigger": "fallback_error"
      }
    ]
  }
}
```

### 按任务选择模型

```json
{
  "model": {
    "router": {
      "enabled": true,
      "rules": [
        {
          "if": "message contains '代码' or '编程' or 'debug'",
          "use": "claude-4-opus-20250514"
        },
        {
          "if": "message contains '分析' or '比较' or '为什么'",
          "use": "claude-4-sonnet-20250514"
        },
        {
          "if": "message length < 30",
          "use": "claude-4-haiku-20250514"
        },
        {
          "if": "has attachment",
          "use": "gpt-4o-2024-11-20"
        }
      ],
      "default": "claude-4-sonnet-20250514"
    }
  }
}
```

```json
{
  "model": {
    "provider": "anthropic",
    "model": "claude-4-sonnet-20250514",
    "apiKey": "${ANTHROPIC_API_KEY}",
    "maxTokens": 4096,
    "temperature": 0.7,
    "topP": 0.9,
    "system": "你是一个有帮助的AI助手，用中文回复。"
  }

```

```json
{
  "model": {
    "provider": "anthropic",
    "model": "claude-4-sonnet-20250514",
    "tools": {
      "enabled": true,
      "whitelist": [
        "read",
        "write",
        "exec",
        "message",
        "web_search"
      ]
    }
  }
}
```

```json
{
```
