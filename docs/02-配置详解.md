## 快速开始：国内模型配置

### 方式一：硅基流动 (SiliconFlow) - 强烈推荐 🌟

**特点**：
- 聚合模型多（DeepSeek、Qwen、Llama 等）
- API 兼容 OpenAI 格式
- 国内访问速度快
- 有免费额度

#### 1. 获取 API Key

1. 访问 [硅基流动官网](https://siliconflow.cn)
2. 注册账号 → 进入控制台
3. 创建 API Key

#### 2. 配置

```bash
# .env
OPENAI_API_KEY=sk-your-siliconflow-key
OPENAI_BASE_URL=https://api.siliconflow.cn/v1
```

```json
{
  "model": {
    "provider": "openai",
    "model": "deepseek-ai/DeepSeek-V3",
    "apiKey": "${OPENAI_API_KEY}",
    "baseUrl": "https://api.siliconflow.cn/v1",
    "maxTokens": 4096,
    "temperature": 0.7
  }
}
```

> 💡 **模型名称**（在硅基流动中）：
> - `deepseek-ai/DeepSeek-V3` - 最新免费最强
> - `deepseek-ai/DeepSeek-R1` - 推理模型
> - `Qwen/Qwen2.5-72B-Instruct` - 开源大模型
> - `THUDM/glm-4-9b-chat` - 智谱 AI

### 方式二：百度千帆 (ERNIE)

#### 1. 获取 API Key

1. 访问 [百度智能云千帆平台](https://qianfan.baidu.com/)
2. 注册账号 → 开通模型 → 创建应用
3. 获取 API Key 和 Secret Key

#### 2. 配置

```bash
# .env
BAIDU_API_KEY=your-api-key
BAIDU_SECRET_KEY=your-secret-key
```

```json
{
  "model": {
    "provider": "baidu",
    "model": "ernie-4.5-8k",
    "apiKey": "${BAIDU_API_KEY}",
    "secretKey": "${BAIDU_SECRET_KEY}"
  }
}
```

#### 可用模型

| 模型名称 | 说明 |
|---------|------|
| `ernie-4.5-8k` | ERNIE 4.5 主模型 |
| `ernie-4.5-8k-preview` | ERNIE 4.5 预览版 |
| `ernie-3.5-8k` | ERNIE 3.5 |
| `ernie-speed-8k` | 速度优先 |

## 本地模型 (Ollama)

适合追求隐私或无限使用的用户。

### 安装

```bash
# macOS
brew install ollama

# Linux
curl -fsSL https://ollama.ai/install.sh | sh
```

### 下载模型

```bash
ollama pull qwen2:1.5b      # 轻量
ollama pull qwen2:3b        # 中等
ollama pull llama3.2:3b     # Llama
```

### 配置

```json
{
  "model": {
    "provider": "ollama",
    "model": "qwen2:1.5b",
    "url": "http://localhost:11434"
  }
}
```

## 多模型配置

### 智能路由

```json
{
  "model": {
    "router": {
      "enabled": true,
      "rules": [
        {
          "if": "message contains '代码' or '编程' or 'debug'",
          "use": "deepseek-ai/DeepSeek-R1"
        },
        {
          "if": "message contains '分析' or '比较' or '为什么'",
          "use": "Qwen/Qwen2.5-72B-Instruct"
        },
        {
          "if": "message length < 30",
          "use": "deepseek-ai/DeepSeek-V3"
        }
      ],
      "default": "Qwen/Qwen2.5-72B-Instruct"
    }
  }
}
```

### 自动降级

```json
{
  "model": {
    "fallback": {
      "ernie-4.5-8k": "ernie-3.5-8k",
      "qwen-max": "qwen-plus"
    }
  }
}
```
