# 🏢 飞书接入指南

## 支持的功能

- ✅ 文字消息
- ✅ 图片/文件/语音
- ✅ 卡片消息 (Interactive Card)
- ✅ 群组机器人
- ✅ 文档 API
- ✅ 云空间 API

## 前置要求

1. 飞书企业账号
2. 开发者后台创建应用：`https://open.feishu.cn/`

## 创建飞书应用

### 1. 创建应用

1. 登录 [飞书开放平台](https://open.feishu.cn/)
2. 进入「应用开发」→「创建应用」
3. 填写应用名称和描述
4. 获取 `App ID` 和 `App Secret`

### 2. 添加权限

在应用详情页 → 权限管理，添加以下权限：

**必须权限：**
- `im:message:send_as_bot` - 以机器人身份发送消息
- `im:message:send_as_bot` - 接收消息
- `im:resource:file` - 文件上传下载

**可选权限：**
- `im:chat:readonly` - 获取群信息
- `drive:drive` - 云盘文件管理
- `wiki:wiki` - 知识库管理

### 3. 发布应用

1. 在「版本管理与发布」中创建版本
2. 提交审核并发布到企业

## 配置 OpenClaw

### 安装依赖

```bash
# 飞书渠道通常已内置，检查是否启用
openclaw channels list
```

### 配置应用凭证

```bash
# 设置 App ID
openclaw config set channels.feishu.appId "cli_xxxxxxxxxxxxx"

# 设置 App Secret
openclaw config set channels.feishu.appSecret "your-app-secret"

# 设置 Verification Token（用于 Webhook）
openclaw config set channels.feishu.verificationToken "your-verification-token"

# 设置 Encrypt Key（可选，增强安全性）
openclaw config set channels.feishu.encryptKey "your-encrypt-key"

# 启用飞书渠道
openclaw config set channels.feishu.enabled true
```

### 完整配置

```json
{
  "channels": {
    "feishu": {
      "enabled": true,
      "appId": "cli_xxxxxxxxxxxxx",
      "appSecret": "your-app-secret",
      "verificationToken": "your-verification-token",
      "encryptKey": "your-encrypt-key",
      "autoReply": true,
      "mention": {
        "notifySelf": false
      }
    }
  }
}
```

## 配置 Webhook

### 1. 创建机器人

在群设置中添加「添加群机器人」→ 选择你的应用

### 2. 设置回调地址

在应用后台配置Webhook：
- URL: `https://your-domain.com/webhook/feishu`
- 事件：`im.message.message_created`

> 💡 **提示**：如果使用内网穿透，可用 `ngrok` 或 `cloudflared`

## 发送消息

### 发送文字

```bash
openclaw message send \
  --channel feishu \
  --target "ou_xxxxxxxxxxxxx" \
  --message "你好！"
```

### 发送卡片消息

```json
{
  "message": {
    "msg_type": "interactive",
    "card": {
      "header": {
        "title": {
          "tag": "plain_text",
          "content": "🎉 欢迎使用"
        },
        "template": "blue"
      },
      "elements": [
        {
          "tag": "div",
          "text": {
            "tag": "lark_md",
            "content": "这是**飞书卡片**消息"
          }
        },
        {
          "tag": "action",
          "actions": [
            {
              "tag": "button",
              "text": {
                "tag": "plain_text",
                "content": "点击我"
              },
              "type": "primary",
              "url": "https://open.feishu.cn"
            }
          ]
        }
      ]
    }
  }
}
```

### 发送图片

```bash
openclaw message send \
  --channel feishu \
  --target "ou_xxxxxxxxxxxxx" \
  --media /path/to/image.png
```

## 常见问题

### Q: 消息发送失败？

1. 检查 App ID/Secret 是否正确
2. 确认应用已发布
3. 检查权限是否足够

```bash
# 查看飞书日志
openclaw logs --follow | grep feishu
```

### Q: Webhook 收不到消息？

1. 确认回调 URL 可公网访问
2. 验证 Verification Token 一致
3. 检查应用是否已启用 IM 权限

### Q: 如何获取用户/群组 ID？

- 用户 ID 格式：`ou_xxxxxxxxxxxxx`
- 群组 ID 格式：`oc_xxxxxxxxxxxxx`
- 可以在消息事件中获取

### Q: 机器人回复被忽略？

确保消息是发送给机器人的，或在群组中 @机器人
