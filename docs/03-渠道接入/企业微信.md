# 🏢 企业微信接入指南

## 支持的功能

- ✅ 文字消息
- ✅ 图片/语音/文件
- ✅ 图文消息
- ✅ 模板消息
- ✅ 应用消息推送
- ✅ 接收消息回调

## 前置要求

1. 企业微信管理员账号
2. 企业微信企业 ID (CorpId)
3. 创建自建应用

## 配置步骤

### 1. 获取企业凭证

1. 登录 [企业微信管理后台](https://work.weixin.qq.com/)
2. 进入「我的企业」→「企业信息」
3. 获取 `CorpID`

### 2. 创建自建应用

1. 进入「应用管理」→「自建应用」
2. 创建新应用
3. 获取 `AgentId` 和 `Secret`

### 3. 配置应用

在应用详情页：
- 设置「接收消息」回调 URL
- 获取 `Token` 和 `EncodingAESKey`

### 4. 配置可信 IP

在「开发者接口」→「IP 白名单」中添加服务器 IP

## 配置 OpenClaw

### 基本配置

```bash
# 设置 CorpID
openclaw config set channels.wecom.corpId "wwxxxxxxxxxxxxx"

# 设置 Secret
openclaw config set channels.wecom.corpSecret "your-corp-secret"

# 设置 AgentId
openclaw config set channels.wecom.agentId 1000001

# 设置 Token
openclaw config set channels.wecom.token "your-token"

# 设置 EncodingAESKey
openclaw config set channels.wecom.encodingAesKey "your-encoding-aes-key"

# 启用企业微信
openclaw config set channels.wecom.enabled true
```

### 完整配置

```json
{
  "channels": {
    "wecom": {
      "enabled": true,
      "corpId": "wwxxxxxxxxxxxxx",
      "corpSecret": "your-corp-secret",
      "agentId": 1000001,
      "token": "your-token",
      "encodingAesKey": "your-encoding-aes-key",
      "sync": {
        "users": true,
        "departments": true
      },
      "autoReply": true
    }
  }
}
```

## 发送消息

### 发送应用消息

```bash
openclaw message send \
  --channel wecom \
  --target "userid1" \
  --message "你好！"
```

### 发送群聊消息

```bash
openclaw message send \
  --channel wecom \
  --target "chatid_xxxxx" \
  --message "群通知"
```

### 发送图片

```bash
openclaw message send \
  --channel wecom \
  --target "userid1" \
  --media /path/to/image.jpg
```

### 发送文件

```bash
openclaw message send \
  --channel wecom \
  --target "userid1" \
  --file /path/to/document.pdf
```

## 接收消息

消息回调格式示例：

```json
{
  "msgtype": "text",
  "agentid": 1000001,
  "text": {
    "content": "用户消息"
  },
  "from_user_name": "张三",
  "from_user_id": "zhangsan",
  "create_time": 1234567890,
  "msg_id": "msg_id_xxxxx"
}
```

### 处理文本消息

```javascript
// skills/wecom-handler/index.js
export default {
  name: "wecom-handler",
  async onMessage(message) {
    const { content, from_user_id } = message;
    
    // 关键词回复
    if (content.includes("帮助")) {
      return "帮助信息：\n1. 查询天气\n2. 预约会议";
    }
    
    // AI 对话
    return await this.ai.reply(content);
  }
}
```

## 常见问题

### Q: 消息发送失败？

1. 检查 CorpID、Secret、AgentId 是否正确
2. 确认应用已发布
3. 检查用户是否在应用可见范围内

```bash
# 查看企业微信日志
openclaw logs --follow | grep wecom
```

### Q: 回调验证失败？

1. 确认 Token 和 EncodingAESKey 与管理后台一致
2. 检查服务器时间是否准确
3. 验证回调 URL 可公网访问

### Q: 如何获取 UserID？

1. 在企业微信中查看用户详情页 URL
2. 使用通讯录 API 获取
3. 在消息事件中获取

### Q: 应用收不到消息？

1. 确认应用已发布
2. 检查应用可见范围
3. 验证回调 URL 配置正确
