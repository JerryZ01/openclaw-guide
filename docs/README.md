## 学习路径

### 路径一：快速入门
1. 阅读 [快速开始](01-快速开始.md)
2. 选择一个渠道接入（如 [Telegram](03-渠道接入/Telegram.md)）
3. 尝试 [基础机器人示例](../examples/basic-bot/)

### 路径二：生产部署
1. 完成后阅读 [配置详解](02-配置详解.md)
2. 学习 [Docker 部署](05-部署运维/Docker部署.md)
3. 配置 [监控告警](05-部署运维/监控告警.md)

### 路径三：深度定制
1. 学习 [技能开发](04-高级功能/技能开发.md)
2. 探索 [多代理路由](04-高级功能/多代理路由.md)
3. 构建自己的应用

## 常见问题

### Q: 如何选择渠道？
- 个人使用：Telegram / WhatsApp
- 企业使用：飞书 / 企业微信
- 社区运营：Discord

### Q: 需要付费吗？
- OpenClaw 核心开源免费
- AI 模型可能产生费用

### Q: 如何获得帮助？
- [GitHub Issues](https://github.com/JerryZ01/openclaw-guide/issues)
- [OpenClaw Discord](https://discord.com/invite/clawd)
