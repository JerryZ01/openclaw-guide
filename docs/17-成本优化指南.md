## 提示优化

### 精简提示词

```diff
- "请帮我查看今天的天气情况，然后考虑一下是否需要带伞，如果是晴天就告诉我可以出去散步，如果是阴天就提醒我带伞，如果是下雨天就建议我待在家里。"
+ "今天天气如何？需要带伞吗？"
```

### 使用系统提示控制输出

```markdown
# 系统提示
你是一个简洁的助手。回答要求：
- 简短直接，不超过 2 句话
- 用中文回复
- 不要过度解释
```

### 限制响应长度

```json
{
  "model": {
    "maxTokens": 500,
    "temperature": 0.7
  }
}
```

## 多模型组合策略

### 智能路由

```json
{
  "model": {
    "router": {
      "enabled": true,
      "rules": [
        {
          "if": "message contains '代码' or '编程' or 'debug'",
          "use": "deepseek-ai/DeepSeek-R1"
        },
        {
          "if": "message contains '分析' or '比较' or '为什么'",
          "use": "Qwen/Qwen2.5-72B-Instruct"
        },
        {
          "if": "message.length < 50",
          "use": "deepseek-ai/DeepSeek-V3"
        }
      ],
      "default": "deepseek-ai/DeepSeek-V3"
    }
  }
}
```

### 自动降级

```json
{
  "model": {
    "fallback": {
      "Qwen/Qwen2.5-72B-Instruct": "Qwen/Qwen2.5-14B-Instruct",
      "deepseek-ai/DeepSeek-R1": "deepseek-ai/DeepSeek-V3"
    }
  }
}
```
