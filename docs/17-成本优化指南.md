## 缓存策略

### 启用缓存

```json
{
  "cache": {
    "enabled": true,
    "ttl": 3600,
    "types": [
      "weather",
      "calendar",
      "system"
    ]
  }
}
```

### 手动缓存频繁查询

```javascript
// 在技能中实现缓存
const cache = new Map();

async function getWeatherCached(city) {
  const key = `weather:${city}`;
  if (cache.has(key)) {
    const cached = cache.get(key);
    if (Date.now() - cached.time < 1800000) { // 30分钟
      return cached.data;
    }
  }
  const data = await fetchWeather(city);
  cache.set(key, { time: Date.now(), data });
  return data;
}
```

### 配额检查脚本

```bash
#!/bin/bash
# check-quotas.sh

# Anthropic
ANTHROPIC_USAGE=$(curl -s -H "x-api-key: $ANTHROPIC_API_KEY" \
  "https://api.anthropic.com/v1/usage" | jq '.usage.total_tokens')

# OpenAI
OPENAI_USAGE=$(curl -s -H "Authorization: Bearer $OPENAI_API_KEY" \
  "https://api.openai.com/v1/usage?date=2026-03-03" | jq '.data[0].n_tokens_used')

echo "今日使用:"
echo "  Anthropic: $(($ANTHROPIC_USAGE / 1000000))M tokens"
echo "  OpenAI: $(($OPENAI_USAGE / 1000000))M tokens"
```

## 本地模型 (Ollama)

### 成本优势

- 一次性硬件投入
- 无限 API 调用
- 适合特定场景

### 安装 Ollama

```bash
# macOS
brew install ollama

# Linux
curl -fsSL https://ollama.ai/install.sh | sh

# Docker
docker run -d -v ollama:/root/.ollama -p 11434:11434 --name ollama ollama/ollama
```

### 模型推荐配置

```json
{
  "model": {
    "provider": "ollama",
    "model": "qwen2:1.5b",
    "url": "http://localhost:11434"
  }
}
```

## 多模型组合策略

### 智能路由

```json
{
  "model": {
    "router": {
      "enabled": true,
      "rules": [
        {
          "if": "message contains '代码' or '编程'",
          "use": "claude-4-sonnet"
        },
        {
          "if": "message.length < 50",
          "use": "claude-4-haiku"
        },
        {
          "if": "message contains '分析' or '思考'",
          "use": "claude-4-opus"
        }
      ],
      "default": "claude-4-sonnet"
    }
  }
}
```

### 自动降级

  "model": {

      "claude-4-sonnet": "claude-4-haiku",
      "claude-4-opus": "claude-4-sonnet"

## 成本监控

### 设置预算告警

// skills/budget-alert.js
module.exports = {
  name: "budget-alert",
  async execute(context) {
    const dailyLimit = 10; // 美元
    const currentSpend = await getCurrentSpend();
    
    if (currentSpend >= dailyLimit * 0.8) {
      await context.message.send(
        `⚠️ 预算提醒: 已使用 $${currentSpend.toFixed(2)} / $${dailyLimit}`
      );
    }
  }
};
