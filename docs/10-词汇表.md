# 📖 词汇表

本文档解释 OpenClaw 相关的核心概念和术语。

## 核心概念

### Gateway
OpenClaw 的核心服务，负责连接各渠道与 AI Agent。
- 端口：默认 16784
- 协议：WebSocket + REST API

### Channel (渠道)
消息通道的抽象，目前支持：
- WhatsApp
- Telegram
- Discord
- 飞书
- 企业微信

### Agent (代理)
处理用户消息的 AI 实体，可以是：
- 本地模型
- 远程 API（如 Claude、GPT）

### Session (会话)
用户与 Agent 之间的对话上下文。
- 主会话（main）：共享状态
- 隔离会话（isolated）：独立运行

### Skill (技能)
扩展 OpenClaw 功能的模块，可以自定义消息处理逻辑。

### Memory (记忆)
OpenClaw 的持久化存储系统，用于跨会话记住信息。

### Cron (定时任务)
内置的任务调度系统，支持一次性提醒和周期性任务。

## 技术术语

| 术语 | 说明 |
|------|------|
| WebSocket | 双向实时通信协议 |
| REST API | 基于 HTTP 的接口风格 |
| Webhook | 回调机制，用于接收外部事件 |
| OAuth | 第三方授权协议 |
| Token | 认证凭证 |
| Session | 对话上下文 |
| Context Window | 上下文窗口大小 |
| Temperature | 生成随机性参数 |
| Max Tokens | 最大输出 token 数 |

## 配置术语

| 术语 | 说明 |
|------|------|
| `enabled` | 是否启用 |
| `allowFrom` | 白名单 |
| `requireMention` | 群组中是否需要 @ |
| `bind` | 绑定地址 |
| `port` | 端口号 |

---

## 常见错误

| 错误 | 说明 |
|------|------|
| `Unauthorized` | 认证失败 |
| `Rate Limited` | 请求过于频繁 |
| `Channel Not Found` | 渠道未配置 |
| `Session Not Found` | 会话不存在 |
