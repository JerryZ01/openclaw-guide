## Prometheus + Grafana

### 1. 安装 Prometheus

```yaml
# docker-compose.yml 添加
  prometheus:
    image: prom/prometheus:latest
    ports:
      - "9090:9090"
    volumes:
      - ./prometheus.yml:/etc/prometheus/prometheus.yml
    restart: unless-stopped
```

### 2. 配置 Prometheus

```yaml
# prometheus.yml
global:
  scrape_interval: 15s

scrape_configs:
  - job_name: 'openclaw'
    static_configs:
      - targets: ['openclaw:16784']
```

### 3. Grafana 仪表盘

导入官方仪表盘：https://grafana.com/dashboards

## 告警规则

### 告警配置

```yaml
# alertmanager.yml
route:
  receiver: 'webhook'
  group_by: ['alertname']

receivers:
  - name: 'webhook'
    webhook_configs:
      - url: 'http://your-bot/alert'
```

### 告警规则示例

```yaml
# rules.yml
groups:
  - name: openclaw
    rules:
      - alert: GatewayDown
        expr: up{job="openclaw"} == 0
        for: 5m
        labels:
          severity: critical
        annotations:
          summary: "Gateway is down"

      - alert: HighErrorRate
        expr: rate(openclaw_errors_total[5m]) > 0.1
        for: 5m
        labels:
          severity: warning
        annotations:
          summary: "High error rate detected"
```

## 关键指标

| 指标 | 说明 | 阈值 |
|------|------|------|
| Gateway 进程 | 是否运行 | N/A |
| 响应时间 | API 延迟 | > 5s |
| 错误率 | 失败请求比例 | > 5% |
| 内存使用 | RSS 内存 | > 2GB |
| 会话数 | 活跃会话 | > 100 |

## 常用监控脚本

### 检查脚本

```bash
#!/bin/bash
# check-openclaw.sh

# 检查进程
if ! pgrep -f "openclaw gateway" > /dev/null; then
  echo "ERROR: Gateway not running"
  exit 1
fi

# 检查端口
if ! nc -z localhost 16784; then
  echo "ERROR: Port not listening"
  exit 1
fi

# 检查响应
if ! curl -s http://localhost:16784/health > /dev/null; then
  echo "ERROR: Health check failed"
  exit 1
fi

echo "OK"
```

### 定时监控 Cron

```bash
# 每 5 分钟检查
*/5 * * * * /path/to/check-openclaw.sh
```

## 日志监控

### 错误统计

```bash
# 统计错误
grep -c "error" /tmp/openclaw/openclaw-$(date +%Y-%m-%d).log

# 实时监控错误
tail -f /tmp/openclaw/openclaw-*.log | grep error
```

### 日志告警

```bash
# 关键词告警
tail -f /tmp/openclaw/openclaw-*.log | \
  grep -E "error|warning|critical" | \
  while read line; do
    sendAlert "$line"
  done
```

## 性能优化

### 慢查询分析

```bash
# 开启慢查询日志
openclaw config set logging.slowQueryThreshold 1000

# 查看慢查询
grep "slow" /tmp/openclaw/openclaw-*.log
```
